C Language
C is a general-purpose programming language created by Dennis Ritchie at the Bell Laboratories in 1972. It is a very popular language, despite being old. The main reason for its popularity is because it is a fundamental language in the field of computer science. C is strongly associated with UNIX, as it was developed to write the UNIX operating system.
SQL Language
SQL stands for Structured Query Language. SQL is a standard language for storing, manipulating and retrieving data in databases. It was developed in the 1970s by IBM researchers Raymond Boyce and Donald Chamberlin. SQL is a domain-specific programming language used for managing and manipulating relational databases.
HTML Language
The first version of HTML was written by Tim Berners-Lee in 1993. Since then, there have been many different versions of HTML. The most widely used version throughout the 2000's was HTML 4.01, which became an official standard in December 1999.

Angular Language
Angular, supported by Google, is an open-source, JS-based software engineering platform for building user interfaces (front-end). Its history traces back to 2009 when Misko Hevery and Adam Abrons, Google engineers, developed the framework known as AngularJS and officially released it in 2010.
C# Language
The C# programming language was designed by Anders Hejlsberg from Microsoft in 2000 and was later approved as an international standard by Ecma (ECMA-334) in 2002 and ISO/IEC (ISO/IEC 23270 and 20619) in 2003. Microsoft introduced C# along with .NET Framework and Visual Studio, both of which were closed-source.
ASP Dotnet Core
Microsoft developed ASP.NET Core, an open-source web framework for building web apps and services, and released it in 2002 as part of version 1.0 of the .NET Framework. ASP.NET Core was originally called ASP.NET vNext, and then ASP.NET 5, but Microsoft changed the name to avoid implying it was an update to the existing ASP.NET framework.
MongoDB Language
MongoDB, Inc. is an American software company that develops and provides commercial support for the source-available database engine MongoDB, a NoSQL database that stores data in JSON-like documents with flexible schemas. MongoDB, Inc. is a member of the MACH Alliance.
React Language
Jordan Walke, a software engineer at Meta, created React in 2011 for Facebook's Newsfeed feature. He initially developed a prototype called "F-Bolt" before renaming it to "FaxJS". React was first released to the public in July 2013, and Walke introduced it to the open-source world in late May 2013.
